How it works

  1. Submit — Open a GitHub Issue with the paper URL
  2. Review — The paper is checked for open-access eligibility
  3. Translate — Full-text translation to Arabic via AI pipeline
  4. Human Review — Scientific and linguistic review by domain specialists
  5. Publish — Arabic version goes live on DAR with attribution

Or contact us directly

To submit a paper: translate@dararchive.com

To volunteer or help: contact@dararchive.com

How can you volunteer?

Language & Translation
  • Arabic language auditing — review translated papers for natural flow, terminology accuracy, and grammar
  • Scientific terminology review — verify domain-specific terms are correctly translated
  • Proofreading final papers before publication
Research
  • Paper scouting — find new open-access papers relevant to DAR's scope
  • License verification — check papers are CC-licensed before processing
  • Author outreach — contact researchers about their translated papers
Technical
  • GitHub contributions — bug fixes, UI improvements, new features
  • SEO — help improve search visibility
  • Tooling — improve the translation and ingestion pipeline
Community
  • Spread the word in academic circles
  • Connect DAR with Arabic-language universities and research centres
